\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_client}{Client}} implements the \mbox{\hyperlink{class_game}{Game}} interface and lives in the \mbox{\hyperlink{class_engine}{Engine}}. In order to play a singleplayer or listening game, the engine instantiates a \mbox{\hyperlink{class_client}{Client}} and a \mbox{\hyperlink{class_server}{Server}}, meaning there are two \mbox{\hyperlink{class_game}{Game}} states running at once. A \mbox{\hyperlink{class_server}{Server}} will dictate game updates to a client, but in other respects they are mostly identical. The \mbox{\hyperlink{class_client}{Client}} does own a \mbox{\hyperlink{class_renderer}{Renderer}} however.
