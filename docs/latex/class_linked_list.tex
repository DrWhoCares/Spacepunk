\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Linked\+List$<$ T $>$}

A \mbox{\hyperlink{class_linked_list}{Linked\+List}} is a type of list that can add and remove nodes anywhere in the list efficiently, and create loops on itself (though this can be dangerous!) Linked lists do not have fast lookup times unlike Array\+Lists, but they are better suited to be queues and other things.
