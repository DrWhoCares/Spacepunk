\begin{DoxyItemize}
\item pathfinder.\+generate\+Simple\+Map(); //\+This updates the pathfinder\textquotesingle{}s copy. This only needs to be done after world generation, and if the terrain ever changes.

\end{DoxyItemize}

There is one pathfinder per world. It provides asynchronous pathfinding.
