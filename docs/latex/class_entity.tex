\doxysubsection{Detailed Description}
An entity is an object that lives in a \mbox{\hyperlink{class_world}{World}}. It is made up of many \mbox{\hyperlink{class_component}{Component}} objects in a tree hierarchy. An entity can move about the world and store unique data about itself as key/value pairs. An entity also has a \mbox{\hyperlink{class_script}{Script}} that defines its unique behavior in the \mbox{\hyperlink{class_world}{World}}.
