\doxysubsection{Detailed Description}
Defines a network interface. If hosting a server, \mbox{\hyperlink{class_net_a30facfaf69c1ae91c0d3685a79fa0176}{host()}} should be called first, followed by \mbox{\hyperlink{class_net_aa8cf78cd34dabe21627ef5d441947213}{connect()}} to connect to clients. If acting as client, \mbox{\hyperlink{class_net_a3aad4ae8eba5053455a6b85d5dc1753f}{term()}} should be called to close any existing connection(s), followed by \mbox{\hyperlink{class_net_aa8cf78cd34dabe21627ef5d441947213}{connect()}} to join a server.
