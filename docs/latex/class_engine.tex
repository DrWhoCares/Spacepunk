\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_engine}{Engine}} object is the root of the entire engine. Here lives top level data about the current engine state\+: the local \mbox{\hyperlink{class_client}{Client}} and \mbox{\hyperlink{class_server}{Server}} (if they are running), functions to step everything forward, update devices, etc.
