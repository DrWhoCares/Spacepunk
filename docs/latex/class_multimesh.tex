\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_multimesh}{Multimesh}} is a unique kind of \mbox{\hyperlink{class_entity}{Entity}} \mbox{\hyperlink{class_component}{Component}} that allows you to combine multiple meshes into a single draw call. This allows you to build a room for instance out of multiple small meshes that get optimized for the engine.
