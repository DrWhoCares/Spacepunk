\hypertarget{index_intro_sec}{}\doxysection{Introduction}\label{index_intro_sec}
Spacepunk is a simple 3D engine that is easy to use and is controlled with the Lua script language.

Some good places to start exploring is the \mbox{\hyperlink{class_engine}{Engine}} class, \mbox{\hyperlink{class_world}{World}} class, \mbox{\hyperlink{class_entity}{Entity}} class, and \mbox{\hyperlink{class_component}{Component}} class.

Recommended tab size when viewing the engine code directly is 4.
