\subsubsection*{template$<$typename T, bool stream = false$>$\newline
class Resource$<$ T, stream $>$}

The \mbox{\hyperlink{class_resource}{Resource}} class provides a way for the \mbox{\hyperlink{class_engine}{Engine}} to cache assets on an as-\/needed basis, and delete them when too much data is consumed. Use the \mbox{\hyperlink{class_resource_aa7eb4c25cf79073b95ae1fba0f79e417}{data\+For\+String()}} method to get an asset out of the \mbox{\hyperlink{class_resource}{Resource}}.
