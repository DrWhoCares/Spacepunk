\doxysubsection{Detailed Description}
A container for a gui (ie, a window) When a frame\textquotesingle{}s size is smaller than its actual size, sliders will automatically be placed in the frame. \mbox{\hyperlink{class_frame}{Frame}} objects can be populated with \mbox{\hyperlink{class_field}{Field}} objects, \mbox{\hyperlink{class_button}{Button}} objects, other \mbox{\hyperlink{class_frame}{Frame}} objects, and more.
