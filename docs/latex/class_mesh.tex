\doxysubsection{Detailed Description}
Meshes are the raw 3D assets loaded from disk that populate a game \mbox{\hyperlink{class_world}{World}}. Do not confuse a \mbox{\hyperlink{class_mesh}{Mesh}} with a \mbox{\hyperlink{class_model}{Model}}. Meshes are unique and represent the raw data itself; a \mbox{\hyperlink{class_model}{Model}} is an \mbox{\hyperlink{class_entity}{Entity}} \mbox{\hyperlink{class_component}{Component}} that binds a \mbox{\hyperlink{class_mesh}{Mesh}} to an \mbox{\hyperlink{class_entity}{Entity}}.
