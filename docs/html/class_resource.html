Detailed Description

template<typename T, bool stream = false>
class Resource< T, stream >

The Resource class provides a way for the Engine to cache assets on an as-needed basis, and delete them when too much data is consumed. Use the dataForString() method to get an asset out of the Resource. Template type must implement Asset

finds and returns the data with the given name from the cache if the data already exists in the cache, it is returned and error is set to ERROR_NONE if the data does not exist in the cache, the resource creates it anew and error is set to ERROR_NOTCACHED if the data failed to be created then error is set to ERROR_CACHEFAILED if the data is currently being streamed then error is set to ERROR_CACHEINPROGRESS
