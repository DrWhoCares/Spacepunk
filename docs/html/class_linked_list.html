Detailed Description

template<typename T>
class LinkedList< T >

A LinkedList is a type of list that can add and remove nodes anywhere in the list efficiently, and create loops on itself (though this can be dangerous!) Linked lists do not have fast lookup times unlike ArrayLists, but they are better suited to be queues and other things.
