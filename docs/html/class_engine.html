Detailed Description

The Engine object is the root of the entire engine. Here lives top level data about the current engine state: the local Client and Server (if they are running), functions to step everything forward, update devices, etc.

enum Engine::msg_t

log message codes

Enumerator
MSG_INFO 

only printed when NDEBUG is undefined

MSG_WARN 

for when you have an unimportant message to report

MSG_ERROR 

for when the game has encountered a small problem

MSG_CRITICAL 

for when the game has encountered a definite problem

MSG_FATAL 

for when the game likely will not continue past this point

MSG_NOTE 

for when you have to exit() immediately

MSG_CHAT 

like info, but more important (not for errors!)

MSG_TYPE_LENGTH 

for when a chat message was received over the net
